# PHP compile
Shell script for compiling latest [PHP](https://php.net) from source.

## Usage
```shell
cd /usr/local/src
git clone https://github.com/Fleshgrinder/php-compile
sh php-compile/compile.sh
```

## Executables
The CLI, FPM, and PEAR executables are provided.
